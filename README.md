# Evaluacion 1 Front End 

Nombre de Proyecto: Intranet de Gestion de Bodega - Hospital Regional de Copiapo

Descripcion del negocio: El Servivio de Copiapo es la unidad logistica encargada de la recepcion, almacenamiento y distribucion de insumos medicos. Opera en el primer piso (Bodega) y segundo piso (Abastecimiento) del edificio pricipal, garantizando el suministro continuo de materiales a todas las areas clinicas y administrativas del recinto hospitalario.

Necesidad digital identificada
El cliente es JUAN NAVARRO, jefe de area de Abastecimiento necesita una intranet de gestion interna que permita:
- Controlar el stock de productos
- Registrar entradas y salidas de productos
- Detallar cada salida: que producto sale, a que servicio se entrega, a quien se le entrega y quien lo entrega.
